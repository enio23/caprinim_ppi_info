# CAPRINIM PPI INFO
PPI Information for CArdiac PRotein INteractions In Mice (CAPRINIM)

This repo contains the information displyed in the CAPRINIM web-application in tabulated formats.

The CAPRINIM web-application can be accessed at: https://shiny.dieterichlab.org/app/caprinim

The full AlphaFold3 evaluation models can be accessed on [Zenodo](https://shiny.dieterichlab.org/app/caprinim)
